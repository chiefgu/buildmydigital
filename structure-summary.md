================================================================================
DESIGN SYSTEM PATTERNS
================================================================================

LAYOUT:
  • Primary: Flexbox (all 9 sections)
  • Container: 7 boxed sections, 2 full-width (hero + footer)
  • Max Width: Boxed sections contained within max-width container
  • Full Width: Hero and final section span full viewport

ANIMATIONS:
  • Entrance animations on logo section
  • Parallax background on team/awards section
  • Likely scroll-triggered animations throughout

SECTION TYPES:
  1. Full-screen Hero
  2. Logo strip
  3. Text + Stats
  4. Value proposition
  5. Portfolio grid
  6. Methodology + Metrics
  7. Parallax feature
  8. Content cards
  9. Final CTA

CONTENT PATTERNS:
  • Headers: Large, bold headlines
  • Subtext: Descriptive paragraphs
  • Stats: Numbers with labels
  • CTAs: Button links
  • Images: Portfolio items, background images
  • Logos: Trust indicators

SPACING:
  • Vertical spacing between sections
  • Internal padding within boxed containers
  • Grid gaps for portfolio items
